# Medicare

## Overview
Medicare is an AI-powered emergency healthcare platform.

## Tech Stack
- Next.js 16
- FastAPI
- Supabase
- TypeScript
- Tailwind CSS
- Gemini AI

## Setup

1. Clone the repository:
git clone https://github.com/your-username/medicare.git

2. Install dependencies:
npm install

3. Configure environment variables.

4. Run the frontend:
npm run dev

5. Run the backend:
uvicorn app.main:app --reload
